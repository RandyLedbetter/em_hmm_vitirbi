Problem Statement

Given a structure of the Hidden Markov Model (figure on the right), and a sequence of observations, consisting of Heads and Tails which are generated by two coins, one is balanced and the other is loaded, our task is to learn the transition and sensory probabilities using the EM for HMM algorithm and then to find the most probable sequence of states (Balanced or Loaded) that generated these observations using Viterbi's algorithm.

Instructions

This application will take initial parameters, in the form of two input files and a single value for N in the form below. It will then serve those initial parameters as input to the EM for HMM algorithm. The output of that process will then be served as input to Vertibi's algorithm, and ultimately, Specificity, Sensitivity, and Accuracy will be calculated by comparing actual state sequence values against learned state sequence values. Finally, a ROC analysis will then be performed, based on changes to the value of N (the number of iterations) with the results of that analysis being presented to the user on the screen.
