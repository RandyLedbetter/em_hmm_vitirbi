Upload Sensory and Transition Probabilities File
Select a comma or space delimited file from your computer file system. It should represent a sequence sensory and transition probabilities. The length of the sequence must be equal to 8, and the eight probabilities must be in the following order: P(B|B), P(B|L), P(L|L), P(L|B), P(H|B), P(T|B), P(H|L), and P(T|L). For example, a correctly formatted input file would look like this:
0.7, 0.3, 0.6, 0.4, 0.5, 0.5, 0.9, 0.1       OR
0.7 0.3 0.6 0.4 0.5 0.5 0.9 0.1
