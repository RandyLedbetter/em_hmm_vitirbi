Upload Observation Sequence File
Select a comma or space delimited file from your computer file system. It should represent a sequence of Heads and Tails coin toss results. Heads = 1 and Tails = 0. Therefore, the file should contain a sequence of 1's and 0's.The length of the sequence must be equal to the value of N that you specified on the previous screen. For example, a correctly formatted input file of size N = 6 would look like this:
1, 0, 0, 1, 1, 0       OR
1 0 0 1 1 0
